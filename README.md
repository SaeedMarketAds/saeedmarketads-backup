# Saeed MarketAds - منصة التسويق الذكي

هذا المشروع مخصص لإدارة الحملات التسويقية الآلية وتحليل البيانات باستخدام الذكاء الاصطناعي.

## المميزات:
* **Saeed DataBot**: بوت ذكي للتفاعل مع العملاء وتحليل البيانات.
* **التسويق بالعمولة**: دعم منصات مثل AliExpress و SHEIN و Noon.
* **واجهة Streamlit**: لوحة تحكم سهلة الاستخدام تعمل عبر المتصفح.

## كيفية التشغيل:
1. تأكد من تثبيت المكتبات الموجودة في ملف `المتطلبات.txt`.
2. قم بتشغيل التطبيق عبر Streamlit باستخدام الملف الأساسي.
